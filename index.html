AI & Machine Learning Project

One Piece Character Image Classification

Proyek Deep Learning untuk klasifikasi gambar karakter anime One Piece menggunakan Convolutional Neural Network (CNN). Model dilatih menggunakan dataset berisi 11.737 gambar dengan total 18 kelas karakter untuk mengenali karakter berdasarkan fitur visual dari gambar.

Python TensorFlow CNN Deep Learning Image Classification Computer Vision
View Project GitHub

Bitcoin Time Series Forecasting

Eksperimen dilakukan menggunakan beberapa arsitektur Deep Learning seperti baseline LSTM dan Seq2Seq LSTM untuk multistep forecasting. Model dibangun menggunakan TensorFlow dan difokuskan pada akurasi prediksi serta stabilitas forecasting.

Python TensorFlow Seq2Seq LSTM Forecasting
View Project GitHub

Sentiment Analysis of SeaBank Application Reviews

Proyek analisis sentimen ulasan aplikasi SeaBank menggunakan beberapa arsitektur Deep Learning yaitu LSTM, GRU, dan CNN for Text. Dataset diproses melalui tahapan preprocessing NLP seperti cleaning text, tokenization, dan stopword removal sebelum dilakukan pelatihan model menggunakan TensorFlow untuk klasifikasi sentimen positif dan negatif.

Python TensorFlow NLP LSTM GRU CNN for Text Sentiment Analysis
